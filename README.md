# trivia-game-python
A beginner-friendly Python trivia game with multiple categories and custom question sets.

A beginner-friendly Python trivia game that includes:
General trivia,
Football trivia,
Three custom trivia sets where the user can add their own questions,
Score tracking,
Randomized questions,
Support for text answers and number answers

This was one of my first Python projects and i designed to show my learning process.

How to Run -
Install Python 3.10 or newer,
Download trivia_game.py,
Run:
python trivia_game.py

Features - 
Random question selection
Real-time score updating
Custom question sets
Uses simple loops, lists, and conditionals

Purpose -
This project is meant to show my early Python learning journey.

Future Improvements -
Save custom questions to a file
Add difficulty levels
